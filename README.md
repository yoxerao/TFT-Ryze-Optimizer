This tool helps you find the best 5 team compositions maximizing active traits, built around **Ryze**.
## How to Run
1.  Clone this repo and ensure you are in the root project directory.
2.  Run:
    ```bash
    npm run dev
    ```
3.  Open your browser to: [http://localhost:5173](http://localhost:5173)
## Features
-   **Team Size Slider**: Adjust from **9 to 20** units.
-   **Emblem Selector**: Search and add origin emblems.
-   **Optimization**: Click **OPTIMIZE** to generate 5 distinct team options.
    -   **Ryze** is always included.
    -   **Priority**: Maximizing distinct **Origin Traits** (e.g., Bilgewater, Ionia, etc.).
    -   **Secondary**: Total Active Traits count.
    -   **Emblem Focus**: A "Bonus Option" (purple) appears which tries to use all your emblems.
